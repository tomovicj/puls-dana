# PulsDana

**PulsDana** is a full-stack application that collects latest news from various portals. It allows users to view news summaries, vote on news items (up-vote, down-vote), and comment on them. News older than 7 days will be archived.


## Technologies

- **Backend:** Spring Boot
- **Frontend:** React
- **Database:** MongoDB


## Features

- Collects the latest news from multiple sources
- Displays news summaries to users
- Voting system for news (up-vote, down-vote)
- Commenting on news articles
- Archiving news older than 7 days
